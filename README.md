# 💳 AI Credit Risk Scoring App

An end-to-end Machine Learning application that evaluates credit risk and supports loan approval decisions.

## 🚀 Live Demo
👉 https://credit-risk-ai-livduk4ttdjaeyvx8vl5h6.streamlit.app/

## 🧠 Features

- Predicts probability of default
- Credit risk scoring system (0–100)
- Automated loan approval decision
- Model explainability (feature impact)
- Model comparison (Logistic Regression vs Random Forest)
- ROC Curve visualization

## ⚙️ Tech Stack

- Python
- Pandas
- Scikit-learn
- Streamlit
- Matplotlib

## 📊 Models

- Logistic Regression
- Random Forest (selected based on ROC AUC)

## 💡 Business Impact

This system simulates how fintech companies assess credit risk to:

- Reduce default rates
- Improve loan approval decisions
- Automate underwriting processes

## 🧠 Key Insight

The target variable was engineered using domain-inspired heuristics, which explains the high model performance.
